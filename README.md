ParqMate is an IoT-based smart parking system designed to enhance parking efficiency through real-time monitoring, automated gate control, and seamless cloud integration. The system utilizes an ESP32 microcontroller, IR sensors for vehicle detection and slot monitoring, and a servo motor for intelligent gate operation based on slot availability. It continuously tracks parking occupancy using multiple sensors, updates vehicle entry and exit counts, and transmits data to the cloud via ThingSpeak for real-time visualization. The backend, supported by Python and MongoDB Atlas, handles data processing, storage, and API integration, while the frontend provides user and admin dashboards with live synchronization of parking data. This project demonstrates effective integration of embedded systems, cloud computing, and full-stack development, highlighting the importance of coordinated implementation across hardware, software, and data layers to build a reliable and scalable smart parking solution.
